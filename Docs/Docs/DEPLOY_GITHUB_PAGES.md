# Publish These Pages With GitHub Pages

## 1) Create a GitHub repository
- Name suggestion: `football-conquerors-site`
- Visibility: public (simplest for Steam URL checks)

## 2) Upload this `docs` folder
- Upload the full `docs` directory from this project.

## 3) Enable Pages
- GitHub repo -> `Settings` -> `Pages`
- `Source`: `Deploy from a branch`
- `Branch`: `main` (or `master`)
- `Folder`: `/docs`
- Save

## 4) Wait for publish
- In 1-3 minutes you get:
  - `https://<github-username>.github.io/<repo-name>/`

## 5) Paste URLs into Steam
- Privacy Policy URL:
  - `https://<github-username>.github.io/<repo-name>/privacy-policy.html`
- Support Website URL:
  - `https://<github-username>.github.io/<repo-name>/support.html`

## Optional: Notion quick alternative
- Create two public Notion pages (`Privacy Policy`, `Support`).
- Use the public page links in Steam fields.
- Later replace with your GitHub Pages links.
